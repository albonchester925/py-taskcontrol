## Python

* Use pip:
    - `pip install taskcontrol`
    - `sudo python -m pip install taskcontrol`
* Using pip3:
    - `pip3 install taskcontrol`
    - `sudo python3 -m pip install taskcontrol`
    - `sudo python -m pip install taskcontrol` (depending on OS)
